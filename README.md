# Deal-Bot + ML Pipeline

This project combines a Telegram bot with a machine learning pipeline. It runs 24Ã—7 on Render, responds to messages, and performs data analysis on `data.csv`.

## Features
- Telegram bot with webhook

2\. Connect to Render
3\. Add `TELEGRAM_TOKEN` as environment variable (optional)
